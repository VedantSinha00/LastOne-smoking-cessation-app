# LastOne — Smoking Cessation Behaviour Change App

LastOne is a behaviour-change mobile application designed to help young smokers quit through awareness, reflection, and personalised coping tools.

This project was built as a product design and system architecture exercise by a 3-member team including a doctor, psychologist, and product manager.

## Problem

Smoking cessation apps often focus only on streak tracking.
Research shows quitting depends on emotional regulation, social context, and habit patterns.

## Solution

LastOne uses:

- Stage-based quitting system
- SOS coping tools
- Reflection logging
- Behaviour pattern analysis
- Personalised support
- Non-judgmental relapse handling
